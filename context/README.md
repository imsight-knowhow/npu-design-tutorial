# Context 目录

本目录是面向 AI 助手与贡献者的集中知识库，存放与项目协作相关的一切上下文：设计文档、可复用指令、开发日志、任务与计划等。所有子目录均包含各自的 README（中文说明）。

子目录速览：
- `design/` 架构与技术规格
- `hints/` 操作指引与排错技巧
- `instructions/` 可复用提示与命令模板
- `logs/` 开发会话与结果记录
- `plans/` 路线图与实施策略
- `refcode/` 参考实现与代码样例
- `roles/` 角色化系统提示与记忆
- `summaries/` 分析与知识沉淀
- `tasks/` 工作项（working/done/backlog）
- `tools/` 项目专用工具脚本

建议：所有文档以“HEADER”元信息开头（Purpose/Status/Date/Dependencies/Target），并遵循命名示例提高可发现性。

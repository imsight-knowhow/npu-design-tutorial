# NPU Design Tutorial

Tutorial materials for designing and implementing NPUs (Neural Processing Units).

## Overview

This repository collects notes, references, and code to support learning and experimenting with NPU design.

## Language Policy

- This project is **English-first**.
- Some materials may be written in Chinese when helpful; those files will usually include `-cn` or `.cn` in the filename.
  - Examples: `getting-started-cn.md`, `README.cn.md`
